# Query Research Methodology

## Query Gathering Sources

### Priority 1: Google Direct
1. **Google Autocomplete**: Type topic + wait for suggestions
2. **People Also Ask**: Expand all boxes for related questions
3. **Related Searches**: Bottom of SERP
4. **Image Search Refinements**: Query bubbles in image search

### Priority 2: Tools
1. **Google Keyword Planner**: Volume + related ideas
2. **Ahrefs/SEMrush**: Keyword explorer
3. **AlsoAsked.com**: PAA tree visualization
4. **AnswerThePublic**: Question mapping

### Priority 3: Community
1. **Reddit**: Subreddit discussions
2. **Quora**: Questions about topic
3. **YouTube Comments**: User questions
4. **Forums**: Industry-specific discussions

---

## Query Classification by Intent

### Informational Intent
```
Signals: what, how, why, guide, tutorial, learn
User goal: Understand, learn something
Content type: Blog posts, guides, tutorials
```

### Commercial Intent
```
Signals: best, top, review, comparison, vs
User goal: Research before purchase
Content type: Comparison pages, reviews, listicles
```

### Transactional Intent
```
Signals: buy, price, discount, order, download
User goal: Take action, purchase
Content type: Product pages, pricing pages
```

### Navigational Intent
```
Signals: [brand name], login, official, site
User goal: Find specific website/page
Content type: Homepage, login pages
```

## Query Clustering Strategy

### Step 1: Group by Topic Facet
```
Main Topic: "Semantic SEO"
├── Definition cluster
│   ├── what is semantic seo
│   ├── semantic seo meaning
│   └── semantic seo definition
├── How-to cluster
│   ├── how to do semantic seo
│   ├── semantic seo strategy
│   └── semantic seo implementation
└── Comparison cluster
    ├── semantic seo vs technical seo
    ├── semantic seo vs traditional seo
    └── semantic seo vs content seo
```

### Step 2: Prioritize by Search Volume + Intent
| Priority | Criteria |
|----------|----------|
| P1 | High volume + matches business goal |
| P2 | Medium volume + high relevance |
| P3 | Low volume but important for topical authority |
