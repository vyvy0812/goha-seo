# Search Intent Definitions & Classification

Derived from holisticseo.digital methodology.

## 1. Primary Intent Categories (7-Type Model)

| Intent Type | Description | Keywords/Indicators | Examples |
|-------------|-------------|---------------------|----------|
| **1. Informational** | User wants to learn something; explanations, processes, or facts. | "How to", "What is", "Guide", "Tutorial", "Definition" | "How do I grow tomatoes", "History of SEO" |
| **2. Transactional** | User wants to interact, download, or complete an action (often non-monetary initially). | "Download", "Register", "Sign up", "Watch", "Access" | "SEO checklist download", "Watch cat video" |
| **3. Commercial** | User is investigating products/services with intent to purchase eventually. | "Best", "Review", "Cheap", "Top", "Comparison", "Buy" | "Buy tricycle", "Best SEO tools", "Cheap plants" |
| **4. Navigational** | User wants to go to a specific website or page. | Brand names + "login", "website", "blog" | "Facebook login", "HolisticSEO blog", "YouTube" |
| **5. Brand** | Specific query for a brand, company, or service provider. | [Company Name], [Brand Name] | "McDonalds", "Nike", "SEO Kitchen" |
| **6. News** | User wants current events, latest reports, or timely updates. | "News", "Latest", "Price", "Update", [Current Event] | "Gold price", "Stock market", "Election results" |
| **7. Local** | User wants results in a specific geographic location. | "Near me", [City Name], "Nearby" | "Restaurants nearby", "Dentist in London", "Lawyer NYC" |

## 2. Google Micro-Intent Classification (The "Know-Do-Go" Framework)

Google categorizes all queries into these fundamental desires:

| Micro-Intent | Description | SERP Features / SEO Strategy |
|--------------|-------------|------------------------------|
| **KNOW** | User wants information. | Featured Snippets, Knowledge Graph, Long-form content. |
| **KNOW SIMPLE** | User wants a specific fact (short answer). | Direct Answer Box (0-click). Hard to rank organically. |
| **DO** | User wants to perform an action (often buy/download). | Shopping Ads, Product Pages. Highly competitive organic. |
| **VISIT IN PERSON** | User wants to visit a physical location. | Local Pack (Maps), GMB Profile. |
| **WEBSITE** | User wants to find a specific source. | Sitelinks, Brand Knowledge Panel. |

## 3. Intent-based Content Strategy

| Detected Intent | Recommended Content Format |
|-----------------|----------------------------|
| Details / Informational | Blog Posts, Guides, Wikis, FAQs |
| Commercial Investigation | Comparison Tables, "Best of" Lists, Reviews, Case Studies |
| Transactional / Do | Product Pages, Landing Pages, Tools, Download Gates |
| Local / Visit | Local Landing Pages, GMB Optimization, Store Locators |
| News | News Articles, Live Blogs, Press Releases |

## 4. Self-Correction Questions
*   **Does Google need my site?**
    *   For "Know Simple" (e.g., "Capital of Belgium") -> No, Google answers it.
    *   For "Visit in Person" -> Google Maps dominates.
    *   For "Do" -> Ads often dominate.
    *   **Opportunity**: Complex "Know" queries and specific "Commercial" investigations.
