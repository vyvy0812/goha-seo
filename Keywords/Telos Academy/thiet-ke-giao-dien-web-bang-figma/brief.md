## III. CHI TIẾT OUTLINE

# H1: Hướng dẫn thiết kế giao diện web bằng figma chuẩn responsive rwd
**Guideline viết Mở bài**: Đánh trúng nỗi đau: Giao diện web đẹp là chưa đủ, mà web đó phải Coding được và không bị "vỡ khung" khi lướt bằng điện thoại di động (Responsive). Chèn đoạn block shortcode `[key_takeaways]` tại đây.

## 1. Nguyên lý 12-column grid (lưới 12 cột) trong thiết kế web
**Guideline viết**: Khẳng định bất kì Website chuyên nghiệp (dùng Bootstrap hay Tailwind) đều tư duy theo layout 12 cột. Hướng dẫn cài đặt hệ thống Layout Grid trên Figma: Đặt 12 cột, margin tự do, gutter 20-30px.

## 2. Các bước thiết kế landing page toàn diện trên figma
**Guideline viết**: Chia làm 4 Phase.
### 2.1. Bước 1: cấu trúc 5 phần không thể thiếu của một website
**Guideline viết**: Liệt kê 5 phần Header (Thanh điều hướng), Hero Banner, Feature (Tính năng sản phẩm), Testimonial (Đánh giá) và Footer.
### 2.2. Bước 2: dùng auto layout (wrap) cho các thành phần
**Guideline viết**: Bắt buộc giải thích tính năng "Wrap" siêu mạnh mẽ trong Auto layout mới của Figma. Nó giúp Box chứa ảnh tự động rớt xuống dòng khi thu hẹp kích thước frame.
### 2.3. Bước 3: biến thể desktop thành giao diện mobile (responsive)
**Guideline viết**: Hướng dẫn tư duy ép cột: Cứ 12 cột trên Desktop (PC) thì ứng với 8 cột trên Tablet (Ipad) và 4 cột trên Mobile. Che giấu các element không cần thiết trên mobile để tránh rườm rà.
### 2.4. Bước 4: xuất file và handoff cho designer
**Guideline viết**: Dùng chế độ Dev mode để Developer vào copy mã màu HEX và CSS Box-shadow. Khuyên xuất SVG thay vì PNG/JPG cho Logo và Icon để tránh bể hạt.

## 3. Khoá học thiết kế web và làm chủ design system tại Telos
**Guideline viết**: Khuyến khích tham gia Mentor 1-on-1 tại Bootcamp UI/UX Telos để được chỉ tay bắt việc cách tổ chức File Web cực lớn, xử lý hàng trăm trang Web nội bộ. Ở đây bạn không chỉ học vẽ Tool mà học Web Vitals (Tốc độ tải trang, UX Writing). Chèn Link.

## FAQ (Câu hỏi thường gặp)
### Figma có cung cấp chức năng hosting xuất thành web chạy thật không?
**Guideline viết**: Trả lời: Mặc định là Không. Figma sinh ra CSS/HTML mẫu, nếu muốn thành web thật (Publish) luôn có thể dùng Plugin đồng bộ như Framer hoặc Webflow.
### Cách tìm template layout web miễn phí trên figma?
**Guideline viết**: Tới Figma Community, gõ từ khoá "Landing page UI Kit free" rồi duplicate về thực hành.

**Guideline viết**: Tóm tắt lại: Dù vẽ web gì hãy nhớ 1 chữ: Responsive. Không được chộp giật mà phải đưa phần tử vào Auto Layout.
