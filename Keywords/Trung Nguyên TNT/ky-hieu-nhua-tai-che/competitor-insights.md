# Competitor Insights: Ký hiệu nhựa tái chế

## Top 3 Analysis
1.  **Levu.vn**: Focuses on "Safety for Food". Very consumer-centric. Good breakdown of 1-7.
2.  **Beligroup**: "Industrial Production" angle but still generic. Lists applications in construction/society.
3.  **Baobiducphat**: "Reuse" focus. Which bottles can be refilled?

## Common Structure (The Baseline)
*   **H1**: Ký hiệu/Bieut tuong nhựa tái chế [An toàn/Ý nghĩa].
*   **Introduction**: Why symbols matter (Recycling/Health).
*   **Core Body**: List of 7 Codes (H3s for each number).
*   **Conclusion**: How to choose safe plastic.

## Content Gaps (The Trung Nguyên Opportunity)
| Competitor Approach | Trung Nguyên TNT "Engineer" Angle |
| :--- | :--- |
| Focus on **Consumption** (Is it safe to eat from?) | Focus on **Production** (How to label it correctly?) |
| Static Images of symbols | **Machinery Context**: How to mold these symbols sharply using CNC/Dispensing tech. |
| Generic advice | **Technical Detail**: Why PVC (Code 3) requires specific dispensing machines to avoid burning/deforming the logo. |

## Strategic Recommendations for Outline
1.  **Adopt the Listicle**: Must have the "7 Codes" section (User expectation).
2.  **Upgrade the Perspective**: Instead of just "PVC is dangerous", say "PVC (Code 3) requires precise processing to be safe and durable - Here is the machine for it."
3.  **Add a "Manufacturing" Section**: "Quy trình gia công logo/nhãn mác nhựa đạt chuẩn quốc tế" (The Missing Link).
