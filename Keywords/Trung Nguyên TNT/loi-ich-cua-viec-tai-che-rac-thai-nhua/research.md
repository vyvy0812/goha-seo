## 3. Brand Context Checklist (Trung Nguyên TNT)
- [x] **Automation Focus**: Emphasize that modern recycling is *automated* (inline granulators + auto-loaders), not manual sorting.
- [x] **Taiwan Quality**: Compare high-quality granulators (sharp blades, low dust) vs. cheap machines (noise, dust).
- [x] **Efficiency**: "Recycling is not just green, it's efficient." (Tái chế không chỉ xanh mà còn là tối ưu chi phí).
- [x] **Product Mapping**: 
    - **Mention**: Máy băm nhựa (Granulator), Máy nghiền (Crusher), Phễu sấy (Dryer - for drying recycled material), Máy hút liệu (Auto-loader - for mixing regrind).
    - **EXCLUDE**: Máy chấm keo PVC (Irrelevant).

## 4. Strategic Recommendations
1.  **Structure**: Start with Environmental Impact (Hook) -> Move quickly to Economic Benefits (The "Meat" for B2B) -> End with How-to (Machinery Solution).
2.  **Voice**: "Industrial Engineer" - use data (e.g., "Retaining 100% material value", "Reducing material cost by 20%").
3.  **Visuals**: Diagrams of a closed-loop system (Injection Machine -> Sprue -> Granulator -> Mixer -> Injection Machine).
