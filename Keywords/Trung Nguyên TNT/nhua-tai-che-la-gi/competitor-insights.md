## Strategic Analysis
*   **Common Angles**:
    *   **Definition**: Both start with "Nhựa tái chế là gì?".
    *   **The 7 Types**: Both strictly follow the 1-7 Resin Code structure (PET, HDPE, PVC, LDPE, PP, PS, Others). This is a mandatory standard.
    *   **Applications**: Both discuss what each plastic is used for.

*   **Content Gaps (Opportunities for Trung Nguyên TNT)**:
    *   **Industrial Depth**: Competitors focus on *consumer* knowledge (what is this bottle?). They lack the *manufacturer's* perspective (how to process it, machine requirements).
    *   **Production Challenges**: No mention of moisture issues, mixing ratios, or contamination handling.
    *   **Automation**: No mention of how machinery enables this process.

*   **Structure Recommendations**:
    *   **H2**: Nhựa tái chế là gì? (Definition)
    *   **H2**: 7 Loại nhựa tái chế phổ biến (Detailed breakdown like competitors)
    *   **H2**: Quy trình sản xuất nhựa tái chế (Step-by-step industrial process - New Value)
    *   **H2**: Nhựa tái chế có an toàn không? (Address user fear)
    *   **H2**: Giải pháp tự động hóa trong ngành tái chế (Trung Nguyên TNT Angle)
