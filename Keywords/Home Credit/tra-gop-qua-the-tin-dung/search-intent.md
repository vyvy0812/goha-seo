# Search Intent Analysis: Trả góp qua thẻ tín dụng

## 1. Phân tích SERP Features

| SERP Feature | Có/Không | Ghi chú |
|---|---|---|
| Featured Snippet | Có | Định nghĩa trả góp qua thẻ tín dụng |
| People Also Ask | Có | FAQ về phí, lãi suất, điều kiện |
| Knowledge Panel | Không | |
| Ads (Google Ads) | Có | Quảng cáo thẻ tín dụng từ ngân hàng |
| Video | Không | |
| Image Pack | Không | |

## 2. Intent Classification (7-Type Model)

| Intent Type | Mức độ | Dấu hiệu nhận biết | Content Vehicle |
|---|---|---|---|
| **Informational** | Cao (Primary) | Snippet, PAA, "là gì", "là sao" | Guide, Definitions, How-to |
| **Commercial** | Trung bình | So sánh hình thức, "cần những gì" | Comparison, Feature list |
| **Transactional** | Thấp | "cách mua", "đăng ký" | CTA, Landing page |

## 3. Micro-Intent Analysis

| Micro-Intent | Keywords liên quan | Content Tone |
|---|---|---|
| **Know** | trả góp qua thẻ tín dụng là gì, là sao, là như thế nào | Giải thích, định nghĩa rõ ràng |
| **Do** | cách trả góp, cách mua trả góp, chuyển đổi trả góp | Hướng dẫn từng bước |
| **Know + Do** | thanh toán trả góp, hạn mức trả góp, cần những gì | Thông tin + hướng dẫn thực hiện |

## 4. Content Format khuyến nghị

- **Dạng bài**: Comprehensive Guide (Hướng dẫn toàn diện)
- **Tone**: Informational + Practical
- **Độ dài tối ưu**: 2500-3000 từ
- **Cấu trúc**: Định nghĩa → Phân loại hình thức → Điều kiện/Hạn mức → Lưu ý → CTA
