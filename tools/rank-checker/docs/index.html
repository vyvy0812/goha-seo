4 Lấy file credentials.json

Bạn cần file credentials.json để kết nối Google Sheets. Có 2 cách:

🅰️ Cách nhanh: Xin từ admin

Nếu được share file credentials.json, chỉ cần đặt vào thư mục rank-dashboard/ → xong.

🅱️ Tự tạo credentials riêng
Tạo Google Cloud Project
Truy cập Google Cloud Console → tạo Project mới
Bật APIs
Menu → APIs & Services → Library
Bật Google Sheets API + Google Drive API
Cấu hình OAuth Consent Screen
APIs & Services → OAuth consent screen
Chọn External → điền tên app → Publish App (để ai cũng dùng được)
Tạo OAuth Client ID
Credentials → Create Credentials → OAuth client ID
Type: Desktop app → Create
Tải file JSON
Nhấn ⬇️ tải JSON → đổi tên thành credentials.json → đặt vào rank-dashboard/
⚠️
Bảo mật: Không upload credentials.json và token.json lên GitHub.

7 Chạy lần đầu & Xác thực Google

npm start

Truy cập: http://localhost:3000

🔐 Xác thực Google Sheets (chỉ cần 1 lần)
Mở Settings trên Dashboard
Nhấn ⚙️ Settings → thấy "⚠ Chưa xác thực"
Click "Xác thực Google Sheets"
Browser mở trang đăng nhập Google
Đăng nhập & cho phép quyền
Chọn tài khoản Google → nhấn Allow / Cho phép
Tự động hoàn tất
Google tự redirect về Dashboard → ✅ Xác thực thành công!
File token.json được tạo tự động. Các lần sau không cần xác thực lại.
✅
Sau khi xác thực, mở lại Settings sẽ thấy "✅ Đã xác thực". Bạn có thể bắt đầu Sync và Check rank!

10 Xử lý lỗi

LỗiGiải pháp
Cannot find moduleChạy npm install
CAPTCHA liên tụcCài WARP 1.1.1.1 → bật WARP
Không đọc được SheetKiểm tra GOOGLE_SHEET_ID trong settings.json
Token expiredXóa token.json → chạy lại → xác thực lại
Port 3000 đã dùngPORT=4000 npm start
Auth loop ("Chưa xác thực")Xóa token.json → restart server → thử lại
